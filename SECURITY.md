# Security Policy

## Supported Versions

The following table lists the versions of **peachy-bot** that are currently supported with security updates:

| Version | Supported          | Notes                       |
| ------- | ------------------ | --------------------------- |
| 5.1.x   | :white_check_mark: | Latest stable release       |
| 5.0.x   | :x:                | No longer maintained        |
| 4.0.x   | :white_check_mark: | LTS (Long Term Support)     |
| < 4.0   | :x:                | Unsupported, please upgrade |

If you are using an unsupported version, we strongly recommend upgrading to a supported release to receive security updates and bug fixes.

## Reporting a Vulnerability

If you discover a security vulnerability in **peachy-bot**, please help us keep the community safe by following these steps:

1. **Do not disclose the vulnerability publicly.**
2. **Report it privately** by emailing us at [security@peachybot.app](mailto:security@peachybot.app) or by opening a security advisory on our [GitHub repository](https://github.com/yuuslokrobjakkroval/peachy-bot/security/advisories).
3. Please include as much detail as possible, including:
   - A description of the vulnerability
   - Steps to reproduce
   - Affected version(s)
   - Any known workarounds

### Response Process

- We will acknowledge receipt of your report within **48 hours**.
- We will investigate and provide updates at least **once per week** until the issue is resolved.
- If the vulnerability is confirmed, we will work with you to coordinate a fix and public disclosure.
- If the report is declined, we will provide a clear explanation.

Thank you for helping us keep **peachy-bot** secure!
